Description
===========
The 'processFile.bal' will read a sample csv file using record I/O API and will write the content to another text file

Prerequisites
=============

The "sample.csv" file is located in the directory $Ballerina_HOME/samples/csvFileProcessor/files/ hence ensure that
"processFile.bal" is executed from $Ballerina_HOME/bin directory

How to run this sample
======================

bin$ ./ballerina run ../samples/csvFileProcessor/processFile.bal
